git status - отображает текущее состояние
git add - подготовка файлов перед коммитом (добавляет файлы в стейдж)
git commit -m <message> - создание коммита с записью изменений
git log - показывает все коммиты до последнего сделанного
git checkout <commit hash>/<branch name> - переход в определенную версию проекта по хэшу/имени ветки (копирует файлы из репозитория в рабочую область, переносит HEAD в эту ветку/коммит)

BRANCH
git branch (-m, -d) <branch name> - создание ветки (переименование, удаление)
git checkout -b <branch name> - создание и переход на ветку 
git switch (-c) <branch name> - переход на ветку (без восстановления файлов)
